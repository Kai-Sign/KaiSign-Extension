# Support

## Bug reports & feature requests
Use [GitHub Issues](https://github.com/Kai-Sign/KaiSign-Extension/issues).
Pick the appropriate template:
- **Bug report** — unexpected popup behavior, decoding errors, etc.
- **Protocol request** — request ERC-7730 metadata for a new contract
  or protocol.

## Security issues
**Do not open a public issue for security vulnerabilities.** Use the
GitHub Security Advisory flow described in [SECURITY.md](SECURITY.md).

## Questions
Open a regular issue with the `question` label.
